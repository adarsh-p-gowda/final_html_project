Why HTML

Hyper text markup language


  • HTML (HyperText Markup Language) is the basic language used to create webpages. It tells the browser how to structure content like headings, paragraphs, images, links, tables, and forms using tags.
  • HTML is easy to learn and is the first step for anyone starting web development‚Äîit works together with CSS (for design) and JavaScript (for interactivity) to build complete websites.

What i had learned

  • HTML uses tags to define content. Most tags have an opening and closing form.
  • An element is a complete HTML structure: opening tag + content + closing tag.
  • Attributes give extra information about a tag.
  • HTML has headings from h1 to h6 for titles and subtitles.
